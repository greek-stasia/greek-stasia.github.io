Welcome to my terminal-styled portfolio! This site showcases my projects, write-ups, and thoughts on software engineering and security.

## About This Site

This portfolio is built with [Hugo](https://gohugo.io/) using the [hugo-theme-shell](https://github.com/Yukuro/hugo-theme-shell) theme, which provides a unique terminal-style interface. You can interact with the site using terminal commands!

### Available Commands

- `help` - Show available commands
- `ls` - List available sections
- `cd <section>` - Navigate to a section
- `clear` - Clear the terminal

## What You'll Find Here

- **Projects**: My software projects and experiments
- **Write-ups**: Technical documentation and CTF write-ups
- **Blog**: Thoughts on security, reverse engineering, and software development
